<h3>⚙️ Installation:</h3>

To run this project locally, you can follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/prayer-on/project-3.git
```

2. Select the Backend folder:
```bash
cd Backend
```

3. Install the dependencies:
```bash
npm install
```

4. Start the server:
```bash
npm start
```
